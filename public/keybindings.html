LemonCUC 按键映射 / Keybindings

Mod = Alt（Super/Win 键会被浏览器拦截）

窗口管理 / Window

快捷键 功能
Alt+Shift+Q 关闭窗口 / Kill window
Alt+H / J / K / L 切换焦点（左/下/上/右）/ Focus
Alt+Shift+H/J/K/L 移动窗口 / Move window
Alt+F 全屏 / Fullscreen
Alt+B 水平分割 / Split horizontal
Alt+V 垂直分割 / Split vertical
Alt+Shift+Space 浮动切换 / Toggle floating
Alt+Space 浮动/平铺焦点切换 / Focus mode toggle
Alt+A 聚焦父容器 / Focus parent

布局 / Layout

快捷键 功能
Alt+S 堆叠布局 / Stacking
Alt+W 标签页布局 / Tabbed
Alt+E 切换分割 / Toggle split

工作区 / Workspaces (1–9)

快捷键 功能
Alt+1~9 切换到工作区 1–9 / Switch workspace
Alt+Shift+1~9 移动窗口到工作区 / Move to workspace

调整大小 / Resize

快捷键 功能
Alt+R 进入调整模式 / Enter resize mode
（模式中）H/J/K/L 缩小宽/增高/缩高/增宽
Esc / Enter 退出调整模式 / Exit resize

启动 / Launch

快捷键 功能
Alt+Return 打开终端 (xterm) / Terminal
Alt+D dmenu 启动器 / Launcher
Alt+C Google Chrome

i3 控制 / i3 Control

快捷键 功能
Alt+Shift+C 重载配置 / Reload config
Alt+Shift+R 重启 i3 / Restart i3
